# PTDS.js
Public Transport Decision Support D3.js Frontend

## Instructions
Clone and install dependencies

    $ git clone https://github.com/plannerstack/PTDS.js
    $ yarn

Run development server with autoreload and linting

    $ yarn start

Build distribution

    $ yarn build
